# Creatives Catalog

Внутренний инструмент CAS.AI для управления рекламными креативами: единый каталог, автонейминг, связка с аналитикой.

## Статус: Discovery / Research

## Проблема

Креативы (видео, баннеры, playables) управляются вручную: дизайнеры сдают → координатор переименовывает → заливает на сети. Нейминг ломает аналитику в Superset.

## Ключевые решения

- Сеть НЕ в нейминг (источник из MMP)
- Сезонность — фиксированная позиция в имени
- Старые креативы не переименовывать
- Открыт вопрос: полный каталог vs бот + Google Drive
