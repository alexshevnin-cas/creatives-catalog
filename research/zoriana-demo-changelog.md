## Перечень изменений в спеке (v1.0 → v1.1)

### Сезонность убрана из нейминга
- Правило 6 (сезонность — фиксированная позиция) удалено
- `{seasonal}` убран из шаблонов и примеров (`V003UE_...` → `V003_...`)
- Таблица сезонных маркеров (STD/UE/NY/EA) удалена
- Поле `seasonal_tag` в модели Creative заменено на `tags` (string[])
- Сезонность теперь — один из тегов (`seasonal`), а не часть имени

### Теги вместо сезонности
- Новое поле `tags` в модели Creative: `gameplay`, `mislead`, `UGC`, `seasonal` и др.
- UI: колонка «Сезон» → «Теги», кликабельные бейджи
- Фильтры: «сезон» → «теги»
- Форма: убран дропдаун «Сезонность», добавлен мульти-селект тегов

### Google Drive → собственный сервер
- Drive заканчивается (~100 ГБ + ~130 ГБ на подходе)
- Rendition: `drive_file_id` → `file_path`, `drive_thumbnail_url` → `thumbnail_path`
- Thumbnails генерируются на сервере (Pillow/ffmpeg), не из Drive API
- Структура хранения: `/storage/creatives/{game}/{type}/...`

### Фазирование уточнено
- V1.0: хранилище на сервере + каталог + автонейминг + теги (заменяет Google Drive)
- V1.2: автозаливка на рекламные сети через API (Meta, Google, TikTok, Unity)
- V2+: аналитика по креативам (spend/ROAS, mislead detection, retention по когорте)

### Пользователи расширены
- CMO: управляет UA-командой, может оперативно найти любой креатив
- UA-менеджер: добавлен как отдельная роль (сам построил прототип каталога, имеет креды сетей)

### Новые решения (D9–D11)
- D9: Каталог, не бот — однозначно
- D10: Теги вместо сезонности
- D11: Фазирование V1.0 → V1.2 → V2+

### Открытые вопросы обновлены
- Q3 закрыт (Zoriana подтвердила скоуп)
- Q4 обновлён: Vitalii Tereshchenko — посмотреть его прототип, у него креды сетей
